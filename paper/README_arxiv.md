# arXiv Package

This directory contains the arXiv source package for the LiveProof preprint.

Source file:

```text
main.tex
```

Generated package:

```text
arxiv-liveproof-v0.1.0.tar.gz
```

Submit as a TeX/LaTeX source archive. If arXiv requests the top-level file, use:

```text
main.tex
```

Before finalizing submission:

1. Preview the PDF on arXiv.
2. Check that all tables fit.
3. Confirm author name and affiliation.
4. Confirm category, likely `cs.AI` with `cs.LG` cross-list.
5. Confirm that the GitHub release link is correct.

The local machine used to prepare this artifact did not have a TeX compiler in
PATH, so final visual PDF validation must happen on arXiv, Overleaf, or a local
LaTeX install.
